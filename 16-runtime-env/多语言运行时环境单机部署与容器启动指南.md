## 1. 简介

本指南提供在单台服务器（Rocky Linux 9 / Ubuntu 24.04）上快速部署 Go、Java、PHP 运行时环境的标准化步骤，包含**主机原生安装**（适合生产环境）与**Docker 快速启动**（适合开发/测试）两种方式。

> 📎 **前置条件**：基础系统工具（curl/tar/git）与 Docker Engine 已安装。
> 如未准备，请先执行本章 [1.1 基础环境初始化与 Docker 安装](#11-基础环境初始化与-docker-安装) 部分。

### 1.1 基础环境初始化与 Docker 安装

在部署具体的运行时环境之前，需要先准备好基础的系统工具和容器运行环境。

#### 1.1.1 基础工具安装

> 🖥️ **执行节点：单机服务器**

```bash
# ── Rocky Linux 9 ──────────────────────────
dnf makecache -y
dnf update -y
dnf install -y curl wget vim git tar unzip bash-completion chrony ca-certificates

# ── Ubuntu 24.04 ───────────────────────────
apt-get update -y
apt-get upgrade -y
apt-get install -y curl wget vim git tar unzip bash-completion chrony ca-certificates gnupg
```

#### 1.1.2 Docker Engine 安装

> 🖥️ **执行节点：单机服务器**

```bash
# ── Rocky Linux 9 ──────────────────────────
dnf install -y dnf-plugins-core
dnf config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
dnf install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
systemctl enable --now docker

# ── Ubuntu 24.04 ───────────────────────────
apt-get update -y
apt-get install -y ca-certificates curl gnupg
rm -f /etc/apt/sources.list.d/docker.list /etc/apt/sources.list.d/docker-ce.list
rm -f /etc/apt/sources.list.d/docker.sources /etc/apt/sources.list.d/docker-ce.sources
rm -f /etc/apt/keyrings/docker.asc
install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | gpg --dearmor -o /etc/apt/keyrings/docker.gpg
chmod a+r /etc/apt/keyrings/docker.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(. /etc/os-release && echo $VERSION_CODENAME) stable" > /etc/apt/sources.list.d/docker.list
apt-get update -y
apt-get install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
systemctl enable --now docker
```

```bash
# ✅ 验证
docker version --format '{{.Server.Version}}'
# 预期输出：29.x.x（以实际安装为准）
```

## 2. Golang 环境

### 2.1 主机安装（生产环境推荐）

> 目标版本：**Go 1.26.1**（示例版本，请按需替换）

#### 2.1.1 安装步骤（Rocky 9 / Ubuntu 24.04 通用）

> 🖥️ **执行节点：单机服务器**

```bash
# 1. 定义版本变量
GO_VERSION="1.26.1"
GO_OS="linux"
GO_ARCH="amd64"
GO_TARBALL="go${GO_VERSION}.${GO_OS}-${GO_ARCH}.tar.gz"

# 2. 下载并解压（官方二进制包）
curl -fsSL -o "/tmp/${GO_TARBALL}" "https://go.dev/dl/${GO_TARBALL}"
rm -rf /usr/local/go && tar -C /usr/local -xzf "/tmp/${GO_TARBALL}"

# 3. 配置环境变量（系统级）
cat > /etc/profile.d/go.sh << 'EOF'
export GOROOT=/usr/local/go
export GOPATH=/opt/go
export PATH=$GOROOT/bin:$GOPATH/bin:$PATH
EOF

# 4. 创建 GOPATH 目录并生效
mkdir -p /opt/go && chmod 0755 /opt/go
source /etc/profile.d/go.sh
```

```bash
# ✅ 验证
go version
# 预期输出：go version go1.26.1 linux/amd64
```

#### 2.1.2 常用配置（国内加速）

```bash
go env -w GOPROXY=https://goproxy.cn,direct
go env -w GOSUMDB=sum.golang.google.cn
```

### 2.2 Docker 快速启动（开发/测试）

无需安装 Go 环境，直接通过容器编译/运行代码。

```bash
# 1. 创建测试代码
mkdir -p /tmp/go-demo && cd /tmp/go-demo
echo 'package main; import "fmt"; func main() { fmt.Println("Hello Docker Go") }' > main.go

# 2. 启动容器运行（挂载当前目录）
docker run --rm -v "$(pwd)":/usr/src/app -w /usr/src/app golang:1.26.1 go run main.go

# 预期输出：Hello Docker Go
```

## 3. Java 环境

### 3.1 主机安装（生产环境推荐）

> 目标版本：**OpenJDK 21 LTS**（生产主流）或 **OpenJDK 25 LTS**（新特性）

#### 3.1.1 Rocky Linux 9 安装步骤

```bash
# ── Rocky Linux 9 ──────────────────────────
# 安装 OpenJDK 21
dnf install -y java-21-openjdk java-21-openjdk-devel

# ✅ 验证
java -version
# 预期输出：openjdk version "21.x.x" ...
```

#### 3.1.2 Ubuntu 24.04 安装步骤

```bash
# ── Ubuntu 24.04 ───────────────────────────
# 安装 OpenJDK 21
apt-get update
apt-get install -y openjdk-21-jdk

# ✅ 验证
java -version
# 预期输出：openjdk version "21.x.x" ...
```

#### 3.1.3 配置 JAVA_HOME（可选）

```bash
# 写入环境变量
cat > /etc/profile.d/java.sh << 'EOF'
export JAVA_HOME=$(dirname $(dirname $(readlink -f $(which java))))
export PATH=$JAVA_HOME/bin:$PATH
EOF
source /etc/profile.d/java.sh
```

### 3.2 Docker 快速启动（开发/测试）

使用 Eclipse Temurin（原 AdoptOpenJDK）镜像。

```bash
# 1. 创建测试代码
mkdir -p /tmp/java-demo && cd /tmp/java-demo
echo 'public class Main { public static void main(String[] args) { System.out.println("Hello Docker Java"); } }' > Main.java

# 2. 编译并运行
docker run --rm -v "$(pwd)":/app -w /app eclipse-temurin:21-jdk sh -c "javac Main.java && java Main"

# 预期输出：Hello Docker Java
```

## 4. PHP 环境

### 4.1 主机安装（生产环境推荐）

> 目标版本：**PHP 8.3**

#### 4.1.1 Rocky Linux 9 安装步骤

使用 Remi 源获取最新 PHP 版本。

```bash
# ── Rocky Linux 9 ──────────────────────────
# 1. 安装 EPEL 和 Remi 源
dnf install -y epel-release
dnf install -y https://rpms.remirepo.net/enterprise/remi-release-9.rpm

# 2. 启用 PHP 8.3 模块
dnf module reset -y php
dnf module enable -y php:remi-8.3

# 3. 安装 PHP 及常用扩展
dnf install -y php php-cli php-fpm php-mysqlnd php-xml php-mbstring php-opcache php-curl php-zip

# 4. 启动 PHP-FPM
systemctl enable --now php-fpm

# ✅ 验证
php -v
# 预期输出：PHP 8.3.x ...
```

#### 4.1.2 Ubuntu 24.04 安装步骤

使用 Ubuntu 默认仓库安装 PHP。

```bash
# ── Ubuntu 24.04 ───────────────────────────
# 1. 安装 PHP 8.3 及扩展
apt-get update
apt-get install -y php8.3 php8.3-cli php8.3-fpm php8.3-mysql php8.3-xml php8.3-mbstring php8.3-opcache php8.3-curl php8.3-zip

# 2. 启动 PHP-FPM
systemctl enable --now php8.3-fpm

# ✅ 验证
php -v
# 预期输出：PHP 8.3.x ...
```

### 4.2 Docker 快速启动（开发/测试）

启动一个内置 Web Server 的 PHP 容器。

```bash
# 1. 创建测试页面
mkdir -p /tmp/php-demo && cd /tmp/php-demo
echo '<?php echo "Hello Docker PHP"; ?>' > index.php

# 2. 启动容器（映射端口 8080）
docker run --rm -d -p 8080:8000 -v "$(pwd)":/var/www/html -w /var/www/html --name php-test php:8.3-cli php -S 0.0.0.0:8000

# 3. 验证访问
curl http://localhost:8080
# 预期输出：Hello Docker PHP

# 4. 清理
docker stop php-test
```

## 5. 卸载与清理

如果需要移除安装的环境，请执行以下命令：

### 5.1 Golang 清理

```bash
rm -rf /usr/local/go /opt/go /etc/profile.d/go.sh
```

### 5.2 Java 清理

```bash
# Rocky Linux
dnf remove -y java-21-openjdk*

# Ubuntu
apt-get remove -y openjdk-21-jdk*
apt-get autoremove -y
```

### 5.3 PHP 清理

```bash
# Rocky Linux
systemctl stop php-fpm
dnf remove -y php*

# Ubuntu
systemctl stop php8.3-fpm
apt-get remove -y php8.3*
apt-get autoremove -y
```
